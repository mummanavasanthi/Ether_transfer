# Ether Transfer Smart Contract

This project demonstrates the deployment of an Ether Transfer Smart Contract on SecureChain (SCAI) Mainnet using Solidity, Remix IDE, and MetaMask wallet.

## Technologies Used

- Solidity
- Remix IDE
- MetaMask
- SecureChain Mainnet (SCAI)

## Features

- Deposit Ether into smart contract
- Check contract balance
- Transfer Ether securely
- Owner-based transfer control

## Deployment Process

1. Installed MetaMask wallet
2. Added SecureChain Mainnet network
3. Received SCAI faucet tokens
4. Created Ether Transfer smart contract in Remix IDE
5. Compiled Solidity contract
6. Connected MetaMask with Remix
7. Deployed contract successfully on SecureChain Mainnet
